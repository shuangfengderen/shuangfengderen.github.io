Research Projects


  • Ph.D project     UC Davis Microwave/Millimeter-Wave Research Center
    1. Developed and designed a milli-meter wave reflectometer to measure centimeter-scale density fluctuations on magnetic confinement fusion device TOKAMAK. Tools used are IDL, Matlab, python and CodeV.
    2. Characterized and tested the related electronic boards using a variety of measurement instruments, say, RF source, spectrum analyzer, Network Analyzer, etc. Worked with Altium Design and micro-controller to design a 1-9 GHz electronic system for EAST (Hefei, China) reflectometer device.
    3. Proceeded on analyzing the experimental data measured using the MIR instrument. The focus is to eliminate instrumental artifacts as the reason for causing observed EHO counter-propagation. EHO is the key factor of QH mode, which is a promising operation scenario for future TOKAMAK, say, ITER (International Thermonuclear Experimental Reactor).

  • M.S. project     UC Davis Big Data Visualization and Interface Design Innovation Group
    1. Tried to develop a new method to quantify uncertainties induced with different Dimensional Reduction (DR) methods, say PCA (principle component analysis), MDS (multidimensional scaling), LAMP (local Affine Multidimensional projection), LSP (least squares projection), PLMP (part-linear multidimensional projection), etc. A lot of recently published DR, clustering, and feature space manipulation methods (also applicable to Machine learning) are investigated. The work also involved with implementation of fast algorithm (LSH locality-sensitive hashing) to find nearest neighbors, implementation of different methods read from the papers. Programing languages used are mostly Matlab and python.

    2. Extended development of the correlation-based star coordinate system, with Jellyfish layout and Sankey diagram layout. Since the original high-dimensional space data is categorized into different clusters based on some of their features, different correlation matrix is achieved for different data cluster; thus, each axis of the star coordinate is curved corresponding to its optimized angle from the varied correlation matrix. A sankey diagram applying hierarchical clustering (not the optimized method) to each data cluster is used to assist users with clear look of how attributes correlation changes through different data clusters. This tool can help to reduce features more efficiently. Programing languages used are D3, python, Javascript, html and CSS.
